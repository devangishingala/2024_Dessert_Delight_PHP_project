1)Start the PC
2)Check the Internet connection if not then connect it.
3)Install XAMPP if it is not there via https://www.apachefriends.org/ link.
4)Store our Dessert_Delight folder in C:/ drive ->XAMPP ->htdocs .
5)Then start Apache, MySQL.
In MySQL click on admin and after it put "root" as username and no password required.
6)Create new database named dessert_delight then import our database from folder .
7)Then go on Apache - localhost and then write the path of folder from htdocs.
8)And run it ..
